**Практическая работа №3 — Семантическая HTML5-разметка**

Данный проект демонстрирует использование современных возможностей HTML5 и базовой стилизации с помощью CSS.  
Реализованы:

- Семантическая структура страницы (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`)

- Доступная HTML-форма с валидацией

- Семантическая таблица с `caption`, `thead`, `tbody`, `tfoot` и атрибутами `scope`

- Базовая адаптивная стилизация
